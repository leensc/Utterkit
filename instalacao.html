Instalação

A instalação do Utterkit é rápida e intuitiva, apenas referencie os arquivos a seguir no seu documento. Utilizamos uma variedade de frameworks amplamente reconhecidos, proporcionando flexibilidade e eficiência ao seu desenvolvimento. Confira a lista a seguir:

Framework/Library Tipo Uso Versão Status CDN Acesso
uikit.min.css CSS minificado Grande parte dos layouts e designs são desenvolvidos com o UIkit. 3.21.9 Obrigatório https://cdn.jsdelivr.net/npm/uikit@3.21.9/dist/css/uikit.min.css Download
uikit.min.js JavaScript minificado É utilizado a maioria das funcionalidades dos componentes padrão do UIkit. 3.21.9 Obrigatório https://cdn.jsdelivr.net/npm/uikit@3.21.9/dist/js/uikit.min.js Download
uikit-icons.min.js JavaScript minificado Os ícones do UIkit não são utilizados, ficando a critério do usuário a escolha dos ícones. 3.21.9 Opcional https://cdn.jsdelivr.net/npm/uikit@3.21.9/dist/js/uikit-icons.min.js Download
