Logos

Adicione uma seção dedicada aos parceiros

A seção logos geralmente é dedicada a exibir os logos de empresas, organizações ou parceiros que têm alguma relação com a empresa proprietária do site. Essas relações podem incluir parcerias de negócios, patrocínios, colaborações em projetos, ou até mesmo clientes renomados. A presença desses logos pode servir para transmitir credibilidade, mostrar associações importantes e destacar as redes de apoio ou colaboração da empresa. Essa seção é útil tanto para os visitantes do site quanto para os parceiros envolvidos, fornecendo uma plataforma para reconhecimento e visibilidade mútuos.

Obs: Ao tentar modificar um valor da opção easing do componente Slider qualquer valor será aplicado aos elementos originais, porém será alterado no elemento .uk-slider-items uma função de temporização diferente para os elementos gerados, criando dois estados e não um como esperado. Uma solução é alterar o valor diretamente no arquivo JavaScript para o mesmo declarado no atributo uk-slider.
